# OutBox Consultores — Página de captação

Landing page de captação e detalhamento do Programa de Consultores da OutBox Soluções Digitais.

Tema: **"Tudo passa. A venda não."** Fala com quem já vende e com quem quer começar.

## Stack
- HTML/CSS/JS puro, sem build.
- Servido por nginx (Dockerfile) para EasyPanel.

## Deploy
- EasyPanel (projeto `experts`, app `seller`), build via Dockerfile, porta 80.
- Domínio: seller.outboxgroup.com.br
- Atualizar: `git push` na main → botão Implantar no EasyPanel.
